### Relevant Links

- [Live deployment](https://beta-lyfe.koyeb.app)

### Development

#### Installing Dependencies

```bash
cd webapp
pnpm install
```


#### Running the local setup

```bash
# pwd: ./webapp/

pnpm webapp:dev # to run the webapp
pnpm backend:dev # to run the ad-hoc backend server
```
